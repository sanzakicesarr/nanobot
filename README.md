# 🤖 NanoBot

Lokaler AI-Agent für Telegram. Läuft auf **Ollama** (lokal, null Kosten).
Kein DNS nötig – reines Polling. Tool-System in Entwicklung.

- 💬 Chat mit llama3.1:8b (lokal)
- 🔧 Tool-System (kommt)
- 🚫 Kein DNS nötig (reines Polling)
- 💸 $0 Kosten
